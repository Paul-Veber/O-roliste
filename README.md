# O-rolist
A simple site to search and organise ttrpg games 

Project made in three weeks at the end of my formation with O'clock 

Current features:
- Create Account
- Modify account information
- Create games and invite people 
- Display private information only to the game's members
- Private chat for a game
- Search for games
- Search user
- Send message to user
- Dynamically display new message in chat
